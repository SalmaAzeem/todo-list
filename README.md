### Quick Start

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd todo-list
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   ng start
   ```

4. **Open your browser**
   Navigate to `http://localhost:4200`
